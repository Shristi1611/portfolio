Real-Time Face Detector

A computer vision application that detects human faces in real-time using a webcam feed. The system processes live video frames and highlights detected faces with bounding boxes.

Tech Used: Python, OpenCV, Haar Cascade Classifier

Key Features: Real-time detection, High accuracy, Lightweight model for fast performance.

Invisibility Cloak Effect

A computer vision project that creates an "invisible cloak" illusion by masking a selected color (like red) and replacing it with the background in live video.

Tech Used: Python, OpenCV, NumPy

Key Features: Background capture, Color detection, Real-time video frame processing.
